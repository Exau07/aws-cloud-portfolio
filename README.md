# aws-cloud-portfolio
Ce repository présente un **portfolio professionnel Cloud AWS** construit autour de **labs progressifs**, conçus pour démontrer des compétences que vous découvrirez dans le README.  L’objectif est de refléter le **travail réel d’un Ingénieur Cloud AWS en poste**, dans un contexte **DSI / production**, et non de simples démonstrations théoriques.
